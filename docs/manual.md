# YOLO Platform 使用手冊（給第一次使用的人）

這份手冊用最白話的方式，帶你從零開始訓練出第一個物件偵測模型。
不需要寫程式，跟著步驟操作即可。

> 截圖待補：之後會在每個步驟附上實際畫面。

---

## 0. 第一次使用：安裝

1. 確認電腦已安裝 **Python 3.10 以上**（安裝時記得勾選「Add Python to PATH」）。
2. 在專案資料夾裡，**雙擊 `install.bat`**，等它自動裝好所有東西。
   - 如果你有 NVIDIA 顯示卡，最後它會提示你是否要裝 GPU 版（訓練快很多）。

---

## 1. 啟動平台

**雙擊 `start.bat`**。接著會發生兩件事：

- 跳出**兩個黑色視窗**（一個是後端、一個是前端）。
  - ⚠️ 這是正常的，**不要關掉它們**——關掉等於關閉平台。視窗裡跑的英文訊息不用理會。
- 瀏覽器自動打開操作介面（網址 `http://localhost:8501`）。

用完想關閉平台時，把那兩個黑色視窗關掉即可。

---

## 2. 介面總覽

左側選單有五個功能：

| 功能 | 用途 |
|------|------|
| 📁 資料集管理 | 告訴平台你的圖片和標註放在哪裡 |
| 🏋️ 模型訓練 | 開始訓練模型 |
| 📈 訓練監控 | 看訓練進度、曲線、結果好壞 |
| 🔍 推論測試 | 用訓練好的模型實際試跑圖片 |
| 📦 模型庫 | 管理訓練成果、刪除不要的東西 |

左側下方會顯示你的 **GPU** 和**磁碟用量**。

---

## 3. 準備資料集

你的資料夾需要長這樣（YOLO 標準格式）：

```
我的資料夾/
├── images/   ← 放圖片（.jpg / .png ...）
└── labels/   ← 放標註（每張圖一個同名 .txt）
```

操作：

1. 進入 **📁 資料集管理**。
2. 點「📂 選擇資料夾…」，在跳出的視窗選到你的資料夾。
   - 如果選到的資料夾**沒有圖片**，平台會提醒你重選。
3. 選擇 **train / val 切分方式**：
   - **自動隨機切分**：平台幫你分出一部分當「驗證集」（建議）。
   - **同一資料夾**：訓練和驗證用同一批圖（驗證分數會比較樂觀）。
4. （選填）填類別名稱，例如 `crack, scratch`。
5. 按「**註冊資料集**」。

---

## 4. 訓練第一個模型

1. 進入 **🏋️ 模型訓練**。
2. 最上面選**訓練模式**：
   - **快速試跑**：約十分鐘，先看看有沒有效果。
   - **標準**：一般用途。
   - **精細**：高解析度、長時間訓練，效果最好但最久。
   - **自訂**：完全自己調。
3. 選**基底模型**（新手用 `yolo11n.pt` 即可，最小最快）和剛剛的**資料集**。
4. 按「🚀 開始訓練」。
5. 平台會自動使用 GPU（如果有）。

> 💡 如果跳出「GPU 記憶體不足」的失敗訊息，把 **Batch size** 調小，或勾選
> 「🛡️ 自動 batch」再試一次。這只會讓那次訓練失敗，**不會弄壞系統**。

---

## 5. 看訓練進度與結果

進入 **📈 訓練監控**：

- 上方顯示**狀態**、**進度**、**本次用了哪些參數**。
- 中間有 **mAP / Loss 曲線**（會隨訓練即時更新）。
- 完成後會出現**紅綠燈判讀**：
  - 🟢 表現良好
  - 🟡 尚可，還能更好
  - 🔴 模型還抓不太到目標，建議多加些標註圖片或拉長訓練

**訓練到一半想停**：按「⏹ 取消訓練」，會在當前回合結束後停止，已訓練的權重會保留。
**想接著繼續練**：對已取消／中斷的任務按「▶️ 繼續訓練」。

---

## 6. 用模型試跑圖片

進入 **🔍 推論測試**：

1. 點「📂 選擇資料夾…」選一個有圖片的資料夾。
2. 選**模型**（可選你剛訓練好的）和**信心閾值**。
3. 按「📂 載入資料夾」，再用「⬅️ 上一張 / 下一張 ➡️」瀏覽，按「🔍 推論」看結果。
4. 想一次跑完整個資料夾，按「🔍🔍 推論全部」。
5. 看到模型表現不好的圖，按「📋 複製到 need_to_train」，
   把它收集起來之後補標註、再訓練，模型會越來越準。

---

## 7. 清理空間

進入 **📦 模型庫**：

- 看每個訓練成果佔多少空間，刪掉不要的。
- 管理 `need_to_train` 裡收集的圖片（可單張刪或清空）。
- 刪除資料集時**只會移除平台的登記，不會動到你原本的圖片資料夾**。

---

## 常見問題

**Q：黑色視窗可以關嗎？**
不行，關了平台就停了。用完再關。

**Q：訓練要多久？**
看資料量、模式和有沒有 GPU。「快速試跑」通常十分鐘內，「精細」可能數小時。

**Q：可以給別人用嗎？**
這是本機個人工具，沒有帳號密碼保護，**請勿**把它開放到網路上給別人連。
